# Welcome to Better SaaS

Better SaaS is a modern, production-ready SaaS application template built with Next.js 15 and the latest web technologies. It provides everything you need to build, deploy, and scale your next SaaS application.

## What is Better SaaS?

Better SaaS is a comprehensive starter template that combines the best practices of modern web development with a focus on developer experience and production readiness. It's designed to help you launch your SaaS product faster by providing a solid foundation with all the essential features already implemented.

## Key Features

### 🔐 Authentication & Authorization
- **Multi-provider authentication** with Better Auth
- Email/password and social login (GitHub, Google)
- Role-based access control (RBAC)
- Session management with database persistence
- Password reset and email verification

### 💳 Payment & Billing
- **Stripe integration** with full subscription management
- Multiple pricing plans and billing cycles
- Webhook handling for payment events
- Billing dashboard and invoice management
- Free trial and subscription controls

### 📁 File Management
- **Cloud storage integration** (AWS S3/Cloudflare R2)
- Secure file uploads with validation
- Image processing and thumbnail generation
- File access control and permissions
- Support for multiple file types

### 🌍 Internationalization
- **Multi-language support** with next-intl
- Built-in English and Chinese translations
- Localized routing and content
- Easy to add new languages

### 📱 Modern UI/UX
- **Responsive design** with Tailwind CSS v4

### 📖 Documentation System

- API documentation

### 🔒 Security & Performance
- **Protected routes** with middleware

- Optimized bundle with Next.js 15

- Performance monitoring

### 🧪 Testing Suite
- **Comprehensive testing** coverage
- Unit tests with Jest and React Testing Library

## Getting Started

Ready to build your SaaS application? Follow our comprehensive [Quick Start](/docs/quickstart) guide to get up and running in minutes.

### Quick Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd better-saas
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Set up environment variables**
   ```bash
   cp env.example .env
   ```

4. **Initialize database**
   ```bash
   pnpm db:push
   ```

5. **Start development server**
   ```bash
   pnpm dev
   ```
